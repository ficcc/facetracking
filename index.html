üöÄ Setup Workflow

Step 1: Get the Code

Download the project files from the GitHub repository. You'll need `face_mouse.py`, `setup.bat`, and `launch.bat`.

Download from GitHub

Step 2: Run Automated Setup

Double-click the setup.bat file. This script automatically creates a virtual environment and installs all required Python libraries.

For non-Windows users, run these commands in your terminal:

# Create and activate virtual environment
python -m venv venv
source venv/bin/activate  # On Linux/macOS
# or
.\\venv\\Scripts\\activate  # On Windows

# Install packages
pip install opencv-python dlib numpy pyautogui imutils

Step 3: Download Facial Landmark Model

The script requires a pre-trained model to detect facial landmarks. Download the file, unzip it, and place it in the same folder as face_mouse.py.

  1. Double-click launch.bat to start the application.
  2. Follow the on-screen instructions to calibrate the controls by performing gestures and pressing 'c'.
  3. The application will start **PAUSED**. Use the in-app controls below to navigate.
Key Action
spacebar **Pause / Resume** all controls.
r **Re-center** the neutral mouse point.
q **Quit** the application.

üõ†Ô∏è Configuration & Tuning

Fine-tune performance by editing these variables at the top of the face_mouse.py script.

# --- CONFIGURATION --- #
WINK_HOLD_DURATION = 1.0      # Seconds to hold a wink.
MOUSE_SENSITIVITY = 1.5       # Lower for slower movement, higher for faster.
SCROLL_SENSITIVITY = 0.5      # Adjusts scrolling speed.
DEAD_ZONE_RADIUS = 7          # The radius (pixels) where no mouse movement occurs.

ü§î Troubleshooting

Error building wheel for `dlib`

This means you need build tools. Install CMake and Visual Studio with the "Desktop development with C++" workload.

`can't grab frame` error

Another app (Zoom, Teams, etc.) is using your webcam. Close all other apps or reboot. Also, check Windows privacy settings to allow desktop apps to access the camera.

Mouse movement is erratic

Press the 'r' key to re-center the neutral point. Ensure you are in a well-lit environment and facing the camera directly.
